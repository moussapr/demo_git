# Demo Git

## Présentation
Ce **projet** a pour but de mettre en oeuvre `git`

commande:
``` sh
git init
```

### Créer un commit

pour ajouter les modifications d'un fichier à votre commit:
```sh
git add <nom de votre fichier>
```
pour ajouter ce fichier au `commit` en cours:
```sh
git commit -m "first commit"

```
